What is this?

μDetector is an Intrusion Detection Tool for Microservices written mostly in Python. It relies on proof-of-concept techniques of Intrusion Detection and automates their functioning for Kubernetes deployments. In practice, after the user has provides a configuration, the tool automatically creates a monitoring agent in the desired node(s) of the cluster which is responsible for collecting system calls and transferring them over to the Intrusion Detection module of the tool. This module covers all the stages of anomaly-based intrusion detection and all activity classified as anomalous will trigger alarms in real-time indicating a possible intrusion in the microservices application. The user can interact with the tool and its monitoring capabilities through a command-line interface or a Web Dashboard. The user is also encouraged to add algorithms to the tool.
